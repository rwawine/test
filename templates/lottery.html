Lottery Verification Guide
Steps to Verify Lottery Results:
  1. Get the Seed: The secret seed is revealed after the lottery
  2. Verify Hash: Check that SHA-256(seed) equals the published hash
  3. Reproduce Results: Use the same algorithm with the seed
  4. Compare Winners: Your calculated winners should match the published results
Algorithm Overview:
1. Generate cryptographically secure seed
2. Create SHA-256 hash of seed (published immediately)
3. For each winner position:
   - Combine seed with position index
   - Calculate SHA-256 hash
   - Convert to number and mod by remaining participants
   - Select participant at that index
   - Remove selected participant from pool
