Проверка честности

Шаги для проверки:

  1. Получите семя: Секретное семя раскрывается после розыгрыша.
  2. Проверьте хеш: Убедитесь, что SHA-256(семя) равно опубликованному хешу.
  3. Воспроизведите результаты: Используйте тот же алгоритм с семенем.
  4. Сравните победителей: Ваши рассчитанные победители должны совпадать с опубликованными.

Алгоритм:

1. Генерация крипто-семени
2. SHA-256(семя) -> хеш (публикуется)
3. Для каждого победителя:
   SHA-256(семя + индекс) -> выбор
